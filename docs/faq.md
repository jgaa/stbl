# FAQ

## Why does this project exist?

`stbl` exists to make it quick and easy to build nice static websites from simple source files, with minimal operational overhead.

It was created to solve real web presence needs in a practical way: fast authoring, fast builds, and predictable output.

## Who is behind the project?

The project is developed by **The Last Viking LTD**.

It was originally built for The Last Viking LTD's own web presence requirements, then evolved into a general-purpose static site generator.

## Does `stbl` cost money?

No. `stbl` is free software.

## Can I use `stbl` for commercial websites?

Yes. You can use `stbl` to build commercial sites.

## Can I use the included styles and color schemes for commercial sites?

Yes. The included themes, styles, and color schemes are intended to be usable for commercial sites as well.

## Can I modify the templates and styles?

Yes. Customization is a core design goal. You can override templates, CSS, and theme settings for your own branding and layout needs.

## Is JavaScript required for sites generated by `stbl`?

No. `stbl` is designed to produce fully functional sites without requiring JavaScript.

## Is `stbl` only for blogs?

No. It supports blogs, landing pages, and general content-driven static websites.

## Is the cache required?

No. The cache is optional. You can disable or remove it without affecting correctness.

## Is `ffmpeg` always required?

No. `ffmpeg` is required only when your site uses video processing.
